### TwitchMod
TwitchMod is a modified version of Twitch Android Application.

**Features:**
- BetterTTV and FrankerFaceZ emotes
- Force ExoPlayer2
- Timestamps
- Prevent messages from being removed
- Copy messages by tapping & holding them
- Autoclicker for channel points
- Floating chat
- Video stats
- And more
